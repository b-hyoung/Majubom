🧭 큰 그림 — 우리가 바꿀 수 있는 것 vs 못 바꾸는 것

❌ 칩 구조라 못 바꿈

  • 단일 안테나(1×1) → CSI quotient·HRV 절대값 불가
  • 2.4GHz only → 5GHz 대비 심박 민감도 ~½ (λ 12.5cm)

✅ 우리가 통제 가능

  • 전원 안정 (≥0.5A + 디커플링 + EN RC)
  • 핀 회피 (스트래핑·PSRAM 핀)
  • 안테나 트임 (6mm keepout)
  • 채널 분리 (2.4GHz 1/6/11)

🎯 우리가 진짜 집중할 5가지

⚠️ 배선 gotcha — 이 핀들은 피해라

센서(ToF 등)를 같이 붙일 때 아래 핀에 배선하면 부팅 실패 / 충돌:
GPIO0 ⚡부트 GPIO3 ⚡스트랩 GPIO45 ⚡VDD_SPI GPIO46 ⚡스트랩 IO35/36/37 ⛔R8 PSRAM
✅ CSI 출력용 시리얼: TXD0=GPIO43 RXD0=GPIO44 · USB D−=IO19 / D+=IO20
→ R8(우리 모델)은 IO35·IO36·IO37이 내부 PSRAM에 물려 외부 사용 불가. 나머지 GPIO는 I2C/SPI로 자유롭게.
