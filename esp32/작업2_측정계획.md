## 🏫 측정 환경 — 학교 랩실 (앉음 자세)

```
[랩실 책상]
     [TX 보드]                              [RX 보드]
        │   ────── 1 ~ 1.5m ──────           │
        │                                    │
       책상 끝                              책상 끝
        │              [의자]                │
        │           사람 앉음                │
        │     (가슴이 LOS 사이에 위치)      │
        │                                    │
   가슴 높이 (책 받침대로 맞춤)
```

### 핵심 조건
- 두 보드 사이 거리: **1~1.5m**
- 보드 높이 = 의자 앉은 사람 가슴 높이 (책으로 받침)
- 안테나 면 서로 마주보게
- 의자에 등받이 의지, 정자세

### 자세
- ✅ 등받이 의지, 양손 책상, 양발 바닥, 자연 호흡
- ❌ 다리 떨기, 핸드폰 보기, 한쪽 기울임, 깊은 한숨

## ⚠️ 환경 의존성 — 같은 곳에서 모든 측정

CSI는 **multipath fingerprint**가 방마다 달라 학습 환경 외에선 정확도 급락 (논문 C §6.3).

### 측정 전 결정
- ✅ **한 책상**에서 1명/2명 측정 모두 (위치·셋업 고정)
- ❌ 1명은 랩실, 2명은 카페 같은 분산 측정 금지
- ✅ 가구·물건 위치 측정 중 변경 금지

### 노이즈 대처
- 사람 적은 시간대 (점심, 저녁, 주말)
- 측정 30초 동안 다른 학생 출입 자제 부탁
- 폰 무음 + 비행기 모드 (WiFi 간섭 ↓)
- 에어컨 동일 설정 유지 (가능하면 OFF)

## 🧠 CNN 모델 — 논문 C 구조 (1D Conv × 2 + Dense × 2)

```
입력: (배치, 2채널[진폭+위상], 52 서브캐리어, 2000 시간샘플)
   ↓
1D Conv (64 filters, kernel=10) + BN + MaxPool(4)
   ↓
1D Conv (128 filters, kernel=5) + BN + MaxPool(4)
   ↓
Flatten
   ↓
Dense (256, ReLU) + BN + Dropout(0.5)
   ↓
Dense (128, ReLU) + BN + Dropout(0.5)
   ↓
Softmax (2 클래스: 1명 / 2명+)
```

총 파라미터 약 4.2M.
