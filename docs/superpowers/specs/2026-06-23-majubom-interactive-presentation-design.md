## 슬라이드 (9장)
1. 타이틀 — 마주봄 · 비접촉 낙상 예방
2. 개요 — 3센서(CSI/ToF/mmWave)→서버→대시보드, "임베디드=측정, 서버=판정"
3. 🔴 라이브 통합 시스템 — 대시보드 iframe(8089) 임베드
4. 데이터 흐름·공통 계약 — raw+quality+presence → baseline·z-score → 4단계 (클릭 흐름도)
5. ToF 진행(조우진) — 침상 재실 4×4@400kHz 안정 + 2중 필터 [이미지칸]
6. mmWave 진행(유현기) — IWR6843→Pi5, TLV→보행지표→z-score [이미지칸]
7. CSI 진행 ①(박형석) — HR·호흡·강도(위상 자기상관)+재실(CNN)+z-score+서버연동 [이미지칸]
8. CSI 진행 ②(박형석) — **한계↔향후 토글**: HRV 절대값 단일안테나 불가(논문·실측)→추세 z-score 전환+다중안테나 PoC [이미지칸]
9. 향후 로드맵 — CSI 다중안테나 PoC·30일 베이스라인 / ToF 8×8 / MQTT 무선화
